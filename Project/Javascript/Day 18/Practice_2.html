Difference between var and let keyword

Redeclare of var and let

var
{
    var x = 20;
    var x = 22;
    console.log(x); // 22
}
let
{
    let x = 20;
    let x = 22; // Uncaught SyntaxError: Identifier 'x' has already been declared
}

Reassign of var and let

var
{
    var x = 20;
     x = 22;
    console.log(x);// 22
}
let
{
    let x = 20;
     x = 22;
    console.log(x);// 22
}
